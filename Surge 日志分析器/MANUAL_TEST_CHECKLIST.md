## App 主入口

- [ ] 从 Scripting App 直接启动，显示“暂无日志数据”空状态。
- [ ] 点击“打开”或“打开文件”，可从文件选择器选择有效的 `.log` 或文本日志。
- [ ] 选择有效日志后，显示内存曲线、统计和日志列表，文件路径正确。
- [ ] 取消文件选择；现有内容和状态不变，无崩溃。
- [ ] 选择空文件或不可读文件；保持现有内容和状态，无错误退出。
- [ ] 选择尚未下载的 iCloud 文件；完成下载后正常读取。
- [ ] 内存滑块逐点选择，当前值、时间和对应时段日志同步变化。
- [ ] “重置/按时段”切换结果正确。
- [ ] ERROR、WARN、NOTICE、INFO、DEBUG、VERBOSE、UNKNOWN 和全部筛选结果正确。
- [ ] 分别按消息、模块名、时间文本搜索；清空搜索后恢复原结果。
- [ ] 约 2000 条及更大日志可完成打开、滚动、筛选和搜索。
- [ ] 点击“完成”关闭页面后脚本正常退出。
